# claude-marketplace

Public Claude Code plugin marketplace.

## Install

```
/plugin marketplace add Lanqu/claude-marketplace
```

## Browse plugins

```
/plugin discover
```

## Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| **psych-review** | Multi-perspective code review through 10 cognitive disorder lenses (Paranoid, OCD, ADHD, Hypochondriac, etc.) with structured consensus aggregation. | `/plugin install psych-review@claude-marketplace` |
